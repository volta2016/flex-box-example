Propiedades de los elementos - flex (Hijos) (Al igual que el contenedor - flex, los elementos - flex tienen propiedades interesantes, con las cuales podremos hacer cosas aun más increíbles).


order: - hijo - Con esta propiedad controlamos el orden de los elementos - flex que están dentro del contenedor - flex order: ; (Los elementos - flex pueden ser reordenados con esta simple propiedad sin la necesidad de reestructurar el código HTML..)).

flex-grow - Valor por defecto: 0 - Esta propiedad especifica el factor que determina hasta que punto un elemento - flex crecerá en relación con el resto de los elementos - flex. - Si los elementos - flex tienen el mismo valor para flex-grow, todos van a tener el mismo tamaño, según el contenedor - flex. - Al definir que en el segundo elemento - flex la propiedad flex-grow va a tener un valor de dos, este va a tomar un ancho que equivale a la suma de dos elementos. - Nota: Los números negativos igual aplican para esta propiedad. flex-grow: ; (resumen arriba).
